# Simple Python AI Agent

This folder contains a minimal starter structure for a Python-based AI agent.

## Structure
- `agent.py`: main agent entry point
- `tools.py`: helper functions for external actions
- `memory.py`: simple in-memory conversation storage
- `requirements.txt`: Python dependencies

## Quick start
1. Create and activate a virtual environment:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the agent:
   ```bash
   python agent.py
   ```

You can expand this scaffold by adding API clients, tool calls, or a more advanced memory system.
